## ⚙️ Run Locally

### Clone the repository

```bash
git clone https://github.com/Yusko0o/Portfolio.git
cd Portfolio/\!final
```

### Open in your browser

Simply open `app.py` it's going to create an localhost server for the website:

```bash
# With Python
python app.py
```

Then go to [http://localhost:8000](http://localhost:8000).
